Instalação e Configuração do Proxmox VE
1. Preparação do Pendrive de Instalação
1.1. Baixar a ISO do Proxmox VE no site oficial: https://www.proxmox.com
1.2. Baixar e instalar o Rufus: https://rufus.ie
1.3. Inserir o pendrive no PC e abrir o Rufus
1.4. Selecionar o pendrive, escolher a ISO do Proxmox e clicar em "Iniciar" para gravar a imagem no pendrive

2. Instalação do Proxmox VE no PC
2.1. Inserir o pendrive no computador onde será instalado o Proxmox
2.2. Entrar na BIOS (geralmente teclas como F2, F10, DEL) e configurar a ordem de boot para iniciar pelo pendrive
2.3. Salvar e sair da BIOS
2.4. Na tela inicial, escolher a opção: "Install Proxmox VE (Graphical)"
2.5. Aceitar os termos de uso: "I agree"
2.6. Selecionar o disco onde o sistema será instalado
2.7. Configurar:

País, fuso horário e layout do teclado

Senha do root e e-mail (importante para notificações do sistema)
2.8. Configurar rede automaticamente ou manualmente (se necessário)
2.9. Finalizar instalação e remover o pendrive ao reiniciar

3. Primeiro Acesso e Configurações Iniciais
3.1. Iniciar o sistema e fazer login como root com a senha definida na instalação
3.2. Atualizar o sistema:

bash
apt update && apt upgrade -y
3.3. Instalar interface gráfica (opcional, mas facilita o uso):

bash
apt install xfce4 xfce4-goodies lightdm -y
apt install firefox-esr
3.4. Após instalação, reiniciar:

bash
reboot
4. Pós-instalação e Acesso via Navegador
4.1. Para acessar a interface web do Proxmox, descubra o IP da máquina:

bash
ip a
4.2. Ou acesse localmente:

text
https://localhost:8006/
4.3. Fazer login com usuário root e senha definida

5. Otimização de Armazenamento
5.1. No terminal, unificar espaço disponível do disco:

bash
lvremove /dev/pve/data   # Confirmar com "y"
lvresize -l +100%FREE /dev/pve/root
resize2fs /dev/mapper/pve-root
6. Habilitar Virtualização Avançada (IOMMU)
6.1. Editar o arquivo de configuração do GRUB:

bash
nano /etc/default/grub
Para processador Intel:

Para Intel:
GRUB_CMDLINE_LINUX_DEFAULT="quiet intel_iommu=on iommu=pt pcie_acs_override=downstream,multifunction nofb nomodeset video=vesafb:off,efifb:off"

Para AMD:
GRUB_CMDLINE_LINUX_DEFAULT="quiet amd_iommu=on iommu=pt pcie_acs_override=downstream,multifunction nofb nomodeset video=vesafb:off,efifb:off"

Salvar (Ctrl+O, Enter, Ctrl+X) e atualizar GRUB:

bash
update-grub
6.3. Reiniciar:

bash
reboot
7. Configurar Módulos do Kernel para Passagem de Dispositivos
7.1. Editar módulos:

bash
nano /etc/modules
Adicionar:

vfio
vfio_iommu_type1
vfio_pci
vfio_virqfd
7.2. Salvar e reiniciar



ATENÇÃO FAZER NA SEQUENCIA 7.3 EM DIANTE SOMENTE SE TIVER OUTRO PC PARA ACESSAR, CASO CONTRARIO CONFIGURE A VM PRIMEIRO PASSO 10



7.3. (Opcional) Configurar parâmetros adicionais para IOMMU:

bash
echo "options vfio_iommu_type1 allow_unsafe_interrupts=1" > /etc/modprobe.d/iommu_unsafe_interrupts.conf
echo "options kvm ignore_msrs=1" > /etc/modprobe.d/kvm.conf

8. Bloquear Drivers de GPU do Host
8.1. Editar blacklist:

bash
nano /etc/modprobe.d/blacklist.conf
Adicionar:

text
blacklist radeon
blacklist nouveau
blacklist nvidia
blacklist nvidiafb



8.2. Salvar e reiniciar

8.3
Adding GPU to VFIO

lspci -v(ver tudo)

lspci -nn | grep -i amd/ati (PLACA DE VIDEO AMD)
lspci -nn | grep -i nvidia/ati (PLACA DE VIDEO NVIDIA)
lspci -nn | grep -i USB


Procure o Endereço da sua Gpu Principal

Rodar o Comando

lspci -n -s 01:00

Pegar o numero do ID e Fabricante

Editar arquivo

nano /etc/modprobe.d/vfio.conf

options vfio-pci ids=1002:67ef,1002:aae0,8086:a36d disable_vga=1

Rodar o comando

update-initramfs -u

Reiniciar o servidor.  ( reboot )

9. Preparar ISOs para as VMs
9.1. Baixar ISO do Windows 11 e VirtIO drivers:

Windows 11: https://www.microsoft.com

VirtIO: https://fedorapeople.org/groups/virt/virtio-win/direct-downloads/
9.2. Fazer upload das ISOs no Proxmox:

Acessar Datacenter > Storage > local > Conteúdo > Fazer upload

10. Criar Máquina Virtual (VM)
10.1. No Proxmox, clicar em Criar VM
10.2. Configurações importantes:

Sistema: Adicionar TPM 2.0 (obrigatório para Windows 11)

Disco: Não usar todo o espaço, deixar pelo menos 10% para o host

CPU e RAM: Reservar pelo menos 10% para o sistema host
10.3. Finalizar criação sem iniciar a VM

11. Adicionar Dispositivos Físicos à VM
11.1. Na VM, acessar Hardware > Adicionar > Dispositivo PCI/USB
11.2. Selecionar: mouse, teclado, fone, controle, placa de vídeo, etc.

Atenção: Dispositivos USB são vinculados por ID da porta. Se mudar de lugar, será necessário readicionar
11.3. Para desligar a VM corretamente: clicar em Desligar no painel da VM (isso desliga a VM primeiro e depois o sistema)

12. Instalar Windows 11 na VM
12.1. Iniciar a VM e seguir a instalação do Windows
12.2. Quando pedir drivers, clicar em Carregar driver e selecionar a ISO do VirtIO
12.3. Após instalar o Windows, acessar o disco do VirtIO e executar:

text
virtio-win-guest-tools.exe
Isso instalará todos os drivers necessários (rede, disco, etc.)

13. Teste de Desempenho
13.1. No PowerShell do Windows (como administrador):

powershell
Winsat disk -drive c
Isso testa a leitura e gravação do disco

14.1. Agora é só instalar placa de vídeo, jogos ou fazer customizações adicionais e por aí vai...
